#### For MacOS system you should do either :
- manually copy `./nginx.conf` file to `/tmp/mutual-aid-app/nginx/nginx.conf` and perform for it `chmod +x nginx.conf` 
- execute `./copy-conf-file-to-tmp.sh` script in order to copy `nginx.conf` file to `/tmp/mutual-aid-app/nginx/nginx.conf` automatically. FYI: execute script from current dir
