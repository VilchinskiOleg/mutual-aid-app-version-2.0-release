#### For MacOS system you should do either :
- manually copy `./init-db.sh` file to `/tmp/mutual-aid-app/postgres/init-db.sh` and perform for it `chmod +x init-db.sh` 
- execute `./copy-init-file-to-tmp.sh` script in order to copy `init-db.sh` file to `/tmp/mutual-aid-app/postgres/init-db.sh` automatically. FYI: execute script from current dir
